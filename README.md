# Normalizing Flow Interactive Slider

This project is an **interactive React-based slider** designed to visually and intuitively explain the concept of **Normalizing Flows** for probability distribution estimation. It is ideal for students, educators, and anyone interested in understanding how normalizing flows transform simple distributions into complex ones using invertible mappings.

## 🌊 What are Normalizing Flows?
Normalizing Flows are a family of machine learning models that transform a simple probability distribution (like a Gaussian) into a more complex one through a sequence of invertible and differentiable mappings. They are widely used for density estimation, generative modeling, and probabilistic inference.

This project provides an interactive way to:
- Visualize how data points are transformed through each flow step
- Understand the intuition behind invertible mappings
- Experiment with parameters and see real-time changes

## 🚀 Features
- **Interactive Slider:** Adjust parameters and see how the distribution changes
- **Step-by-step Visualization:** Watch the transformation at each stage
- **Educational Content:** Inline explanations and tooltips
- **Modern UI:** Built with React and Tailwind CSS for a clean, responsive experience

## 📦 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/normalizing-flow-slider.git
   cd normalizing-flow-slider
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:5173` (or as indicated in your terminal).
